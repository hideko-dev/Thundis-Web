# General usage

This is a simple and commonly used method of retrieving data by sending parameters and a request to a URL.
There are two ways to use this API: using user id or using user tag.

```text filename="When using id"
https://thundis.vercel.app/id/XXXXXXXXXXXXXXXXXX
```

```text filename="When using tag"
https://thundis.vercel.app/tag/XXXXXXXXXXXXXXXXXX
```

<Callout type="info">Substitute your Discord account ID after the /id or tag/ and send your request.</Callout>

The data below is as retrieved from the developer; the URL is one below.

